# Mat-chart

A small collection of **Matplotlib commands** for plotting graphs and visualizing data.  
This repository is a learning project where I practice turning raw data into clear, meaningful visuals.

---

## 📊 Features
- Simple examples of line, bar, and scatter plots  
- Step-by-step commands for customizing graphs  
- Easy-to-follow code snippets for beginners  
- Focus on practical data visualization techniques
